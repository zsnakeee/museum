# Requirements
- PHP 8.3

# Instaltion

- Install vendor
```bash
composer install
```

- Make .env from .env.example
```bash
copy .env.example to .env
```

- Generate APP_KEY
```bash
php artisan key:generate
```

- Storage Link
```bash
php artisan storage:link
```

- Cache Icons
```bash
php artisan icons:cache
```

- Run
```bash
php artisan serve
```
